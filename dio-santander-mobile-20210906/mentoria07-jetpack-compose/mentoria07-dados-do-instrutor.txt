Resumo
A branch feature/theme tem um exemplinho de como vocês podem utilizar o Theme (dark e secondary secondary também ;-)).

Importante: Os todos estão sendo gravados em lista, na memória. Desafio: Será que você pode criar um local data source para persistir esses itens? Bora integrar um repository nesse view model? E um network data source para salvar se você ou algum amigo poder fazer uma api?

Como não tinha muito tempo, não conseguimos aprofundar tanto. Então aí vão algumas dicas de assuntos para estudo.
Se acreditar que é muita coisa, faça os estudos ao seu tempo, respeite bastante o seu corpo e a mente - Podes clonar o repo que compartilhei e ficar brincando com o projeto também :)
